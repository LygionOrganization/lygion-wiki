欢迎访问 **Lygion Wiki**。

Lygion Wiki 是 Lygion Robotics / 灵影机器人的产品技术文档中心，面向开发者、工程师、创客和产品集成用户，提供产品使用说明、硬件接线指南、软件开发教程、SDK 示例、开源项目资料以及常见问题说明。

本文档旨在帮助用户快速完成产品上手、硬件连接、参数配置、程序开发和系统集成。

## 如何查找文档

如果您已经购买或正在评估灵影产品，可以通过以下方式查找对应资料：

- 使用页面顶部的搜索栏，输入产品型号、接口类型或功能关键词。
- 进入对应的产品分类，按产品系列查找文档。
- 首次使用灵影产品时，建议先阅读 [快速开始](getting-started.md)，了解基础调试流程、通信接口和常用控制方式。

!!! note "说明"
    灵影多数产品基于 **TTL 总线**、**CAN 总线（暂时还未上架）** 进行设计。通过快速开始文档，您可以快速了解总线设备的连接方式、设备 ID 配置、基础通信流程以及常用调试方法。

## 文档分类

### 总线设备

本分类包含灵影底层控制与通信相关产品的使用说明，适用于需要进行设备接线、总线通信、参数配置和嵌入式集成的用户。

主要包括：

- TTL 总线设备
- TTL 编码器
- TTL 步进电机驱动器
- 节点扩展模块
- ESP32-S3 控制板
- 相关调试工具与通信示例

### 机器人相关模组

本分类包含可用于机器人系统搭建的功能模块，适用于移动机器人、机械臂、实验平台和自动化设备的结构集成与控制开发。

主要包括：

- 机器人关节模组
- 执行器模组
- 驱动轮模组
- 舵轮模组
- 同步轮与编码器模组
- 轮毂电机及相关驱动模块

### 机器人整机

本分类包含灵影完整机器人系统的使用说明和开发资料，适用于希望直接使用整机平台进行二次开发、算法验证或应用集成的用户。

主要包括：

- 机械臂
- 移动底盘
- 复合机器人
- 教学与实验平台
- 整机系统调试与控制示例

### 灵影开源

本分类包含灵影提供的开源软件、SDK、示例工程和系统集成说明，适用于需要进行程序开发、二次集成或上位机控制的用户。

主要包括：

- Python SDK
- C++ SDK
- ROS2 示例
- Web / AI Agent 控制接口
- 开源机器人项目
- 示例代码与开发工具
