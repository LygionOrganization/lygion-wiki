# Lygion Wiki

Welcome to **Lygion Wiki**.

Lygion Wiki is the official technical documentation center for **Lygion Robotics**. It is designed for developers, engineers, makers, educators, and system integrators who use Lygion products in robotics projects.

Here you can find product manuals, hardware wiring guides, software tutorials, SDK examples, open-source projects, and troubleshooting notes.

This documentation is intended to help you quickly understand Lygion products, connect the hardware, configure parameters, run example programs, and integrate the devices into your own robotic systems.

## Finding the Right Documentation

If you have purchased a Lygion product or are evaluating one for your project, you can find the relevant documentation in the following ways:

- Use the search bar at the top of the page to search by product model, interface type, or function.
- Browse the product categories and locate the documentation by product series.
- If this is your first time using Lygion products, we recommend starting with [Getting Started](getting-started.md). It introduces the basic workflow for wiring, communication, debugging, and device control.

!!! note "Note"
    Many Lygion products are designed around bus-based control systems, especially **TTL bus** devices. CAN bus products are also planned.  
    The Getting Started guide explains the basic connection methods, device ID configuration, communication workflow, and common debugging tools used with Lygion bus devices.

## Documentation Categories

### Bus Devices

This section covers Lygion products related to low-level control, communication, and embedded integration.

It is useful if you need to connect devices, configure bus IDs, adjust parameters, communicate with hardware, or integrate Lygion modules into your own controller.

Main topics include:

- TTL bus devices
- TTL encoders
- TTL stepper motor drivers
- Bus node expansion modules
- ESP32-S3 controller boards
- Debugging tools and communication examples

### Robot Modules

This section covers functional modules for building robotic systems.

These modules are suitable for mobile robots, robotic arms, experimental platforms, automation equipment, and custom robotics projects.

Main topics include:

- Robotic joint modules
- Actuator modules
- Drive wheel modules
- Steering wheel modules
- Timing pulley and encoder modules
- Hub motors and related drive modules

### Complete Robots

This section covers complete robot platforms from Lygion.

It is intended for users who want to start development directly from a ready-to-use robotic system, such as for secondary development, algorithm testing, teaching, or application integration.

Main topics include:

- Robotic arms
- Mobile robot chassis
- Mobile manipulators
- Educational and experimental platforms
- System-level setup, debugging, and control examples

### Lygion Open Source

This section contains open-source software, SDKs, example projects, and integration guides provided by Lygion.

It is useful for users who want to develop applications, build host-side control software, integrate Lygion hardware with existing systems, or customize robot behavior.

Main topics include:

- Python SDK
- C++ SDK
- ROS2 examples
- Web and AI Agent control interfaces
- Open-source robot projects
- Example code and development tools
