# 总线设备

本分类包含 Lygion Robotics 的 TTL 总线设备、编码器、驱动器和控制板。

## 产品列表

| 产品 | 简介 |
| --- | --- |
| [TTL Adapter (A)](ttl-adapter-a.md) | 板载 USB-TTL 芯片，可以将电脑的 USB 接口或 MCU 的 UART 转换为单线 TTL 总线通信接口 |
| [TTL Stepper Driver (A)](ttl-stepper-driver-a.md) | TTL 总线 步进电机驱动板，包含多种控制方式 |
| [TTL Encoder E02](ttl-encoder-e02.md) | 12bit 360° 绝对角度磁编码器 |
| [TTL Node (A)](ttl-node-a.md) | TTL 总线节点模块，支持 S.BUS、RGB、PWM 输出 |
| [Robot Driver with ESP32S3 Lite](robot-driver-with-esp32s3-lite.md) | ESP32S3 机器人控制板，支持 TTL / RS485 / CAN |

| [HC-1.25 8P Hub (A)](hc-1.25-8p-hub-a.md) | 板载 5264-3P 接口用于供电，8个 HC-1.25-3P 接口和板间通信接口，用于扩展小电流的总线设备和供电解耦 |
