# Lygion Wiki

Product documentation and developer wiki for Lygion Robotics hardware, modules, SDKs, and open-source robot projects.

## Overview

Lygion Wiki is the official documentation repository for Lygion Robotics products and developer resources.  
It provides technical documentation, getting-started guides, hardware usage instructions, SDK examples, and open-source project references for robotics developers, makers, educators, and system integrators.

This repository is built with [MkDocs](https://www.mkdocs.org/) and is intended to be maintained as a structured, version-controlled documentation site.

## Documentation Scope

- Bus devices and robot control modules
- Robot actuators, drivers, encoders, and motor modules
- Complete robot platforms and open-source robot projects
- Python / C++ SDKs and example programs
- Wiring, communication protocols, and configuration guides
- Product tutorials, FAQs, and troubleshooting notes
